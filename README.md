# Snell 一键安装脚本
## Snell：debian/centos/ubuntu等

```
wget -O snell.sh --no-check-certificate https://git.io/Snell.sh && chmod +x snell.sh && ./snell.sh
```

## Alpine系统：
```
bash <(curl -sSL https://raw.githubusercontent.com/jyucoeng/Snell/main/snell-alpine.sh)
```
